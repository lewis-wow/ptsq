# Server side calls

Server side call also run all the middlewares registered to resolver unlike calling resolver function only.

```ts title="server.ts"
const { middleware, resolver, router, serve } = createServer({
  ctx: createContext,
});

const testMutation = resolver.mutation({
  input: z.object({ name: z.string() }),
  output: z.string(),
  resolve: async ({ input /* { name: string } */ }) /* string */ => {
    return `Hello, ${input.name}`;
  },
});

const baseRouter = router({
  test: testMutation,
});

const caller = baseRouter.createCaller({});

const result = await caller.test.mutate({
  name: /* string */ 'John',
});
```

The `Router.createCaller` function expects context object as the parameter, because there is no context on the server side without request from client side.
